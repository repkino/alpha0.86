# Основные правила для ИИ-ассистента Antigravity

В самом начале каждой сессии, ПЕРЕД любыми другими действиями, ОБЯЗАТЕЛЬНО прочитай файл по абсолютному пути d:\git\alpha0.86\files\Antigravity.txt с помощью инструмента view_file и строго следуй всем инструкциям из него.

### Работа с файлами Construct 3 (СТРОГИЙ ЗАПРЕТ НА ПРЯМОЕ РЕДАКТИРОВАНИЕ)
СТРОГО ЗАПРЕЩЕНО напрямую изменять, перезаписывать или сохранять любые файлы `.json` проекта (особенно листы событий, такие как `_UI.json`, `_scompi.json` и файлы сцен).
Конструктор хранит актуальные листы в памяти редактора. Любая попытка изменить или сохранить файлы напрямую приведет к потере данных или крашу редактора!
Для применения изменений:
1. Объясни пользователю, что нужно изменить.
2. Сгенерируй код в формате **Construct 3 Clipboard Data (JSON)** для копирования.
3. НИКОГДА не используй инструменты редактирования файлов на `.json` файлах проекта.
